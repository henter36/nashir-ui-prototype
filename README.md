# Nashir UI Prototype

واجهة محلية تجريبية لمنصة ناشر.

## الحالة الحالية

هذا المستودع يحتوي على Prototype واجهة فقط، وليس تنفيذًا نهائيًا.

## الصفحات الحالية

- DashboardPage
- StoreSetupPage
- CampaignWizardPage
- ContentStudioPage
- ReviewPage
- SettingsPage

## القيود الحالية

- لا يوجد Backend
- لا يوجد API
- لا يوجد Database
- لا يوجد Auth
- لا يوجد نشر فعلي
- لا يوجد إرسال WhatsApp أو Email
- لا يوجد توليد AI حقيقي
- البيانات محلية وتجريبية فقط
